🔒 API キーの取り扱い
・あなたの OpenAI API キーは、お使いのブラウザのローカルストレージのみに保存されます（他端末への自動同期はしません）。
・API は OpenAI 公式（https://api.openai.com）への通信時のみ使用されます。
・利用料金はあなたの OpenAI アカウントに直接課金されます。
・ 必ず OpenAI ダッシュボードで「Spending limits」を設定してください。万一キーが漏洩した場合の損害を上限化できます。

マイタグ管理

一括シートに出力する「タグ」列の候補を管理します。AI が判定したカテゴリに基づき、該当するタグを商品ページで自動推奨します。
形式: 1 行 1 タグ。タグ名だけで OK（スペース以降は補足扱いで、マッチングは冒頭の単語で判定します）。
例:
時計
時計 DDP　← 冒頭の「時計」で自動推奨対象
ポケカ PSA　← 冒頭の「ポケカ」で自動推奨対象
細かいキーワード制御をしたい場合のみ、タグ名<TAB>keyword1, keyword2 形式も使えます。空行や「#」で始まる行は無視。

商品のエクスポート先を複数登録できます。作業中に簡単に切り替え可能です。
Googleスプレッドシート内のシート名を指定してください
💡 使い方:
1. 複数のスプレッドシートを事前に登録
2. 商品エクスポート時にドロップダウンから選択
3. 前回選択したものが自動的に選ばれます
📋 Google Apps Scriptのコードを表示 
function doPost(e) {
  try {
    // リクエストボディを解析
    const data = JSON.parse(e.postData.contents);
    const values = data.values;
    const sheetName = data.sheetName || 'インポート用'; // デフォルト

    // スプレッドシートを取得
    const spreadsheet = SpreadsheetApp.getActiveSpreadsheet();
    let sheet = spreadsheet.getSheetByName(sheetName);

    // シートが存在しない場合は作成
    if (!sheet) {
      sheet = spreadsheet.insertSheet(sheetName);
    }

    // データを追加
    sheet.appendRow(values);

    return ContentService.createTextOutput(JSON.stringify({
      success: true,
      message: sheetName + 'に追加しました'
    })).setMimeType(ContentService.MimeType.JSON);

  } catch (error) {
    return ContentService.createTextOutput(JSON.stringify({
      success: false,
      error: error.toString()
    })).setMimeType(ContentService.MimeType.JSON);
  }
}
