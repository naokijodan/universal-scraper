プライバシーポリシー

とりこみ君AI Chrome拡張機能

はじめに

本プライバシーポリシーは、Chrome拡張機能「とりこみ君AI」（以下「本拡張機能」）における、ユーザー情報の取り扱いについて説明します。

収集するデータ

本拡張機能は、以下のデータをユーザーのブラウザ内でのみ取得・処理します：

  • ECサイト（eBay、楽天、Amazon、メルカリ、ヤフオク、ラクマ、ハードオフ等）の商品情報（商品名、価格、URL、画像URL、商品説明等）
  • ユーザーが設定した拡張機能の設定値（API キー、モデル、カスタムプロンプト等）
重要：本拡張機能は、ユーザーの個人情報（氏名、住所、クレジットカード情報等）を収集することはありません。

データの使用目的

収集したデータは、以下の目的でのみ使用されます：

  • 商品情報の表示およびクリップボードへのコピー
  • ユーザーが指定したGoogleスプレッドシートへのデータエクスポート
  • OpenAI API を使用した eBay 出品用データ（タイトル・説明・カテゴリ ID・Item Specifics）の生成（AI 翻訳機能を有効にした場合のみ）
  • 拡張機能の設定の保存と復元

データの送信

本拡張機能は、ユーザーの明示的な操作により、以下の送信を行います：

  • ユーザー自身が指定した Google スプレッドシートへの商品データの送信
  • AI 翻訳機能を有効にした場合: OpenAI（https://api.openai.com）への商品データ・メイン画像 URL・プロンプトの送信
重要：本拡張機能の開発者は、ユーザーのデータを収集、保存、または第三者に提供することはありません。すべてのデータ送信は、ユーザー自身の管理下にある Google スプレッドシート、または OpenAI に対してのみ行われます。
OpenAI のデータ取り扱いに関しては、OpenAI のプライバシーポリシーをご参照ください。

OpenAI API キーの取り扱い（AI 翻訳機能）

  • ユーザーが入力した OpenAI API キーは、ブラウザのローカルストレージ（chrome.storage.local）にのみ保存されます。
  • API キーは 他の端末への自動同期は行いません（chrome.storage.sync には保存しません）。
  • API キーの送信先は OpenAI 公式 API（https://api.openai.com）のみです。
  • API キーは chrome.storage.local.setAccessLevel によって信頼されたコンテキスト（拡張機能の Service Worker および設定画面）からのみ読み取り可能に制限されます。
  • 本拡張機能の利用ログ・テレメトリーを開発者を含む第三者に送信することはありません。
  • ユーザーは設定画面からいつでも API キーを削除できます。
  • API の利用料金は、ユーザー自身の OpenAI アカウントに直接課金されます。
セキュリティ免責事項：API キーはお使いの PC のローカルストレージ上に平文相当で保存されます。万一 PC が紛失・盗難、マルウェア感染、第三者アクセスなどの被害に遭った場合、API キーが漏洩する可能性があります。被害を最小化するため、OpenAI ダッシュボードで Spending limits（支出上限）を設定することを強く推奨します。OS のディスク暗号化（FileVault 等）の有効化も推奨します。

データの保存

  • 拡張機能の設定情報（API キー以外）は、chrome.storage.sync に保存され、Google アカウントを通じて同期される場合があります。
  • OpenAI API キーは、chrome.storage.local にのみ保存され、他端末との自動同期は行いません。
  • 保存されたデータは、ユーザーのブラウザ内にのみ存在し、開発者の運用するサーバーには一切送信されません。
  • 拡張機能をアンインストールすると、保存されたデータは削除されます。

第三者への提供

本拡張機能は、ユーザーのデータを第三者に販売、貸与、または提供することはありません。

権限について

本拡張機能は、以下の権限を使用します：

  • activeTab：現在表示中のタブで商品情報を取得するため
  • scripting：ECサイトのページから商品情報を抽出するため
  • clipboardWrite：商品情報をクリップボードにコピーするため
  • storage：拡張機能の設定を保存するため
  • host_permissions：対応ECサイトおよびGoogleスプレッドシートと連携するため

プライバシーポリシーの変更

本プライバシーポリシーは、必要に応じて更新されることがあります。重要な変更がある場合は、拡張機能のアップデート時にお知らせします。
