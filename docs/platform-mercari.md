# メルカリ専用プロンプト（platform_mercari）

**役割**: 共通プロンプト（system_common）の後ろに連結する、メルカリ固有の指示。

---

## プロンプト本体

```
# プラットフォーム情報
- 商品ページの取得元: メルカリ（https://jp.mercari.com）
- 商品データ内の `platform` フィールドが "mercari" または "mercari_shop" の場合、以下の指示が適用されます。

# メルカリ特有のノイズ除外（必ず除外）
日本語の商品説明には、出品者個人の都合や日本市場特有の決まり文句が混在します。
これらは eBay の Title / Description / Item Specifics に含めず、無視してください:

## 出品者個人の都合（除外）
- 「即購入OK」「即決OK」「即購入歓迎」
- 「専用です」「○○様専用」「専用にします」
- 「コメント不要」「コメントなしOK」「ノークレーム・ノーリターン」
- 「値下げ不可」「お値引き不可」
- 「即発送」「即日発送」「翌日発送」「土日のみ発送」
- 「自宅保管品です」「自宅保管のため神経質な方はご遠慮ください」
- 「ペット飼っています」「喫煙者がいます」
- 「すり替え防止のため返品はご遠慮ください」
- 「写真にて判断し、ご納得頂いた方のみお買い求めください」
- 「梱包して発送いたしますが、発送後の郵送中の事故...」（送料・配送に関する免責）

## メルカリ独自の文言（除外）
- 「種別···シングルカード」のような区切り文字を含むカテゴリ表記（情報として参考にしてよいが、Description には書き起こさない）
- 「らくらくメルカリ便」「ゆうゆうメルカリ便」「定形外」などの配送方法名

# 商品本体の情報（保持・翻訳対象）
以下は商品本体の情報なので、翻訳して Description に含める:
- ブランド・型番・モデル名
- 素材・材質
- サイズ・寸法・重量
- 状態（傷・汚れ・付属品の有無）
- 色・柄
- 生産国・製造元
- セット内容
- 動作確認の有無（電化製品の場合）
- ジャンク品・現状品・未確認品などの注意書きは Condition として翻訳する（"Junk", "As-is", "Untested" など）

# メルカリの「商品の状態」マッピング（参考）
メルカリ側の状態 → eBay 相当（itemSpecifics の Condition / Item Condition Description に活用）
- 「新品、未使用」 → "Brand New" / "New"
- 「未使用に近い」 → "New (Other)" / "Like New"
- 「目立った傷や汚れなし」 → "Used - Excellent"
- 「やや傷や汚れあり」 → "Used - Good"
- 「傷や汚れあり」 → "Used - Fair"
- 「全体的に状態が悪い」 → "For parts or not working" / "Used - Poor"

# 画像の活用
- メルカリは商品画像が複数枚ある（1 枚目がメイン）
- メイン画像から以下を判断してよい:
  - キャラクター名・シリーズ名（説明文にない場合）
  - ブランドロゴの確認
  - 状態の確認（傷・汚れ・付属品の有無）
  - 色・素材感
- 画像で確認できない情報を推測した場合は warnings に明記する

# メルカリの DOM 抽出データの読み方
- `title`: 商品タイトル（日本語、商品名に説明的なキーワードが詰まっていることが多い）
- `description`: 商品説明（日本語、上記ノイズが混入している）
- `categoryPath`: メルカリ側のカテゴリパス（例: "ゲーム・おもちゃ・グッズ" > "トレーディングカード" > "ポケモンカードゲーム"）
- `condition`: 商品の状態（上記マッピング参照）
- `seller`: 出品者名（eBay 出力には含めない）
- `itemDetails`: 詳細フィールド（種別、ブランド、シリーズ、配送方法 など）

# Description 構成のメルカリ特化テンプレート（参考）
```html
<p>{商品の魅力を 1〜2 文で要約}</p>

<p><strong>Item Details:</strong></p>
<ul>
  <li>{ブランド/型番/モデル}</li>
  <li>{素材/サイズ}</li>
  <li>{セット内容}</li>
</ul>

<p><strong>Condition:</strong></p>
<p>{メルカリの「商品の状態」を英訳 + 説明文中の傷・汚れ情報を補足}</p>
<p>Please check the photos carefully for details.</p>

<p><strong>From Japan</strong> — All items are shipped from Japan with care.</p>
```
（このテンプレートは目安。商品によって構成は柔軟に変えてよい）
```

## 設計メモ

- このプロンプトは `chrome.storage` に保存される「公式版」。ユーザーが編集すると「ユーザー版」が優先される。
- メルカリショップ（mercari_shop）も同じプロンプトを使う想定。違いがあれば platform_mercari_shop.md を別途作る。
- 楽天・Yahoo・ハードオフ等の他プラットフォームは Phase 3 で同じ構造で追加する。
