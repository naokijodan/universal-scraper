# プラットフォーム情報
- 商品ページの取得元: eBay（https://www.ebay.com）
- 商品データ内の `platform` フィールドが "ebay" の場合、以下の指示が適用されます。

# Item Specifics に必ず含めるべき項目（メルカリ → eBay 共通必須）
以下は商品カテゴリに関わらず eBay 出品で重要なので、可能な限り出力に含めること。値が不明な場合のみ省略してよい（その場合は warnings に記載）。

- **Country/Region of Manufacture**: 商品の製造国。メルカリは日本のフリマサイトなので、日本ブランド・日本製・国内中古品の場合は `Japan` を入れる。海外ブランドの並行輸入品など明確に他国製の場合はその国（例: Switzerland, USA, China）。判断つかない場合は `Japan`(日本のフリマからの出品である事実を優先)にしつつ warnings に「製造国を画像から確認できず、メルカリ出品から推定して Japan と記載」のように明記。
- **Brand**: ブランド名・メーカー名（タイトル / 説明文 / 画像のロゴから判明する場合）。確認できない・ノーブランドの場合は `Unbranded` か省略 + warnings 記載。
- **Type**: 商品の種類（例: Wristwatch, Action Figure, Trading Card, Handbag, Sneakers）。
- **Country of Origin**: 上記 Country/Region of Manufacture と同義として扱う。両方を別キーで出すのではなく `Country/Region of Manufacture` 1 つに統一する。

# 商品カテゴリ別の推奨必須項目（該当カテゴリでは積極的に含める）
- 時計: Brand / Department (Men/Women/Unisex) / Movement / Display / Case Material / Band Material / Dial Color / Case Size / **Wrist Size（腕周り。説明文に「腕周り○○cm」「ベルト長」等の記載があれば必ず Item Specifics に含める。形式は **"○○cm / ○○in"**（cm と inch 併記）。例: "17.5cm / 6.9in"。インチは cm ÷ 2.54、小数1位）** / Water Resistance / Reference Number (型番) / Model
- 衣類・バッグ・靴: Brand / Department / Size / Color / Material / Style
- フィギュア: Brand / Character / Franchise / Material / Size / Year of Manufacture
- カードゲーム（ポケモン等）: Game / Character / Set / Card Number / Language / Manufacturer / Card Condition / Rarity
- 電化製品: Brand / Model / MPN / Type / Color / Power Source / Connectivity
- 本・雑誌: Author / Publisher / Language / Format / Publication Year
- アンティーク・骨董: Original/Reproduction / Vintage / Material / Style / Period

# eBay特有のノイズ除外（必ず除外）

# 商品本体の情報（保持・翻訳対象）
- ブランド・型番・モデル名
- 素材・材質
- サイズ・寸法・重量
- 状態（傷・汚れ・付属品の有無）
- 色・柄
- 生産国・製造元
- セット内容
- 動作確認の有無（電化製品の場合）
- ジャンク品・現状品・未確認品などの注意書きは Condition として翻訳する（"Junk", "As-is", "Untested" など）

# eBayの「商品の状態」マッピング（参考）
（出力時は主観評価語 "Brand New", "Like New", "Mint", "Near Mint", "Excellent" を含めない。eBay 元データに「New」とあっても "Unused" に置き換える）
- "New" → "Unused"
- "New with tags" → "Unused with tags"
- "New without tags" → "Unused without tags"
- "New other" / "Open Box" → "Open Box (Used, very minimal signs of use)"
- "Certified refurbished" → "Certified Refurbished"
- "Seller refurbished" → "Seller Refurbished"
- "Used" / "Pre-owned" → "Used"
- "For parts or not working" → "For parts or not working"

# 画像の活用
- eBayは商品画像が複数枚あり、1 枚目がメイン。
- メイン画像から判断してよいもの:
  - キャラクター名・シリーズ名（説明文にない場合）
  - ブランドロゴの確認
  - 状態の確認（傷・汚れ・付属品の有無）
  - 色・素材感
- 画像で確認できない情報を推測した場合は warnings に明記する。
