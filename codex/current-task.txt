## メルカリリトライ待機時間を4秒→2秒に変更

ファイル: ~/Desktop/とりこみ君/content.js

### 重要ルール
- 既存の冒頭ヘルパー関数（1-101行付近）は変更しない
- console.logは使わない（_logを使う）。ただしconsole.warn/errorは許容

### 修正内容
1275行付近の以下の行：
```javascript
        await new Promise(resolve => setTimeout(resolve, 4000));
```
を以下に変更：
```javascript
        await new Promise(resolve => setTimeout(resolve, 2000));
```

また、同じブロック内の_logメッセージも修正：
```javascript
        _log('⚠️ メルカリ抽出データ不完全（タイトルまたは価格なし）、4秒後にリトライ...');
```
を以下に変更：
```javascript
        _log('⚠️ メルカリ抽出データ不完全（タイトルまたは価格なし）、2秒後にリトライ...');
```

### 確認事項
- 変更はメルカリリトライの待機時間のみ（1268-1295行付近のブロック内）
- 初期待機の4000ms（1186行）は変更しない
- 他の箇所には影響しない
